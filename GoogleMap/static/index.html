Google Maps Scraper

Powered by Python + Selenium

📋 Hướng dẫn sử dụng

  1. Mở Google Maps và tìm kiếm (VD: "công ty tại hà đông")
  2. Copy URL từ thanh địa chỉ trình duyệt
  3. Paste URL vào ô bên dưới
  4. Click "Bắt đầu Scrape" và đợi kết quả (30-60 giây)
  5. Xuất dữ liệu dưới dạng CSV hoặc JSON

📊 Kết quả: 0 công ty

STT Tên công ty Địa chỉ Số điện thoại Đánh giá Link

🛠️ Hướng dẫn cài đặt

Backend (Python):

# Cài đặt dependencies pip install flask flask-cors selenium webdriver-manager # Chạy backend python app.py
⚠️ Lưu ý: Cần cài đặt ChromeDriver tương thích với Chrome của bạn.
Hoặc dùng: pip install webdriver-manager để tự động cài.
